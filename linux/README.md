# Linux Commands, Help & Tips

# Managing Packages

### Package manager

```
$ apt
```

### Package manager update

```
$ apt update
```

### Package manager upgrade

```
$ apt upgrade
```

### Install package

```
$ apt install package
```

### Remove package

```
$ apt remove package
```

# Navigating the File System


### Print working directory

```
$ pwd
```

### Print working directory

```
$ pwd
```

### List directories

```
$ ls

$ ls -a

$ ls -l

$ ls -1 (eg, -1 directories level)
```

### Go in directory

```
$ cd directory_path
```

### Go in root directory

```
$ cd ~
```
